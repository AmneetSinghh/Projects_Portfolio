3. Telegram LRU Cache Operations (JavaScript)

LRU Cache

It is a Backend project which generally shows how the chatting functionality in these messaging applications like telegram/whatsapp works. I implemented LRU Cache using linked-list and default hash-map of JavaScript and showed how we can make our chatting functionality fast by using LRU Cache in that.

  • Basically when the user sends message, all the chats which are above it will shift down and the chats which are below it will shift up. So this functionality is so much time taking but in real-time they do it in mili-seconds. So I represent how LRU-Cache will be useful in this case.
  • Technologies- HTML, CSS and JavaScript Gaming framework Phaser for Frontend and Node js for Backend.
